# MarkDown复杂场景应用  
## 混合强调样式

基本强调样式

- **加粗**
- *倾斜*  
- ~~删除线~~

混合强调样式

- ***加粗倾斜***  
- **~~加粗删除~~**  
- *~~倾斜删除~~*
- ***~~加粗倾斜删除~~***
